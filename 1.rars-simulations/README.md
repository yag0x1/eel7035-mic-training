
# RARS Simulations

## Introduction

This folder contains studies and simulations of basic RISC-V assembly instructions. It focuses strictly on the base integer instruction set and some pseudo-instructions, and does not include the RV32M (Multiply Extension), RV32A (Atomic Extension), RV32F / D (Single / Double-Precision Floating-Point Extensions) and RV32C (Compressed Extension).

---

## Setup and Usage

To run these simulations, RARS (RISC-V Assembler and Runtime Simulator) is required.

### 1. Download
The latest release of RARS can be downloaded directly from the official repository:
🔗 [TheThirdOne/rars GitHub Repository](https://github.com/TheThirdOne/rars)

### 2. Run
Ensure Java is installed. Open a terminal in the directory where the `.jar` file was downloaded and execute the following command:

```bash
java -jar rars1_6.jar
```
